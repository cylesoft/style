So with the "gridded" version, you wrap the page with a div with class "grid-container" instead of "container".

All of the same interface elements are present, and you get to use divs with classes "section" and "group" and "col".

And you size the divs with classes "s1" for 1-wide up to "s12" which is 12-wide (full width).

The only thing that does not work very well is the "aside" class.
